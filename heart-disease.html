Heart Disease Prediction System

Machine Learning & Healthcare Analytics Project

Project Screenshots

Technologies Used

Python · Machine Learning · Logistic Regression · Data Analysis · HTML · CSS

Project Overview

This project predicts the likelihood of heart disease based on patient health parameters such as age, cholesterol, blood pressure, and heart rate. A machine learning model was trained and deployed through a simple web interface to provide real-time predictions.

Key Features

  • Data preprocessing and feature selection for healthcare datasets
  • Logistic Regression model for binary classification
  • User-friendly web interface for real-time prediction
  • Focus on accuracy, interpretability, and healthcare relevance
