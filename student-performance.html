Project Overview

This project analyzes student academic performance using structured datasets to identify trends, strengths, and improvement areas. Interactive dashboards were created to support data-driven decision-making.

Key Features

  • Data cleaning and preprocessing of student datasets
  • Performance comparison using KPIs and analytical charts
  • Subject-wise and overall performance visualization
  • Interactive dashboards for actionable insights
